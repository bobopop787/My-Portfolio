PacMen Factory

      The first project I made was PacMen Factory. In this project, you can create pacmen and animate them.


How to use

      To create pacmen, use any of the Add * Pacmen buttons. To start the animation, click the Start Game button. You can also pause the game when the animation is running using the Pause Game button. You can also remove pacmen using any of the Remove * Pacmen buttons. More info can be found in the readme.md in the GitHub repo.

Eye Exercise

      The second project I made was the Eye Exercise. This project uses JavaScript to detect mouse movement.


How to use

      In this project, simply move around your mouse on the webpage. The eyes will constantly look towards where your mouse is.

Real Time Bus Tracker

      The final project I made was the Real Time Bus Tracker. This project uses MBTA's API to track the locations of buses in Boston.


How to use

      In this project, the red map markers show the current locations of buses in Boston. Note that these markers only update every 20 seconds. When the Show stops between MIT and Harvard button is clicked, the green marker will be animated and show the stops between MIT and Harvard.
